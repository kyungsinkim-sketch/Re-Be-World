개인정보처리방침

시행일: 2026년 7월 19일 · Be.Ark 운영: REBE WORLD FZCO

요약: Be.Ark는 로컬 우선(local-first)으로 설계되었습니다. 대화, 에이전트 기억, Living Files, 가져온 문서는 모두 사용자의 컴퓨터에 저장되며 저희 서버로 업로드되지 않습니다. 저희는 계정·구독·고객지원 운영에 필요한 최소한의 정보만 받습니다.

1. 사용자 기기에만 남는 데이터

Be.Ark의 핵심 기능은 전부 사용자의 컴퓨터에서 동작합니다:

  • 채팅 기록, 에이전트 기억·프로필
  • Living Files 및 가져온 모든 문서 (PDF, DOCX, 카카오톡 대화 내보내기 등)
  • 로컬 AI 모델(내장 Ollama 런타임)이 처리하는 내용 — 완전 오프라인
  • 사용자가 연결한 API 키·서비스 토큰 (기기 내 로컬 볼트에 저장)

위 데이터는 Be.Ark 서버로 전송되지 않습니다. 앱 데이터 폴더 삭제 또는 앱 제거 시 함께 삭제됩니다.

2. 서버에서 수집하는 데이터

  • 계정: 계정을 만들기로 선택한 경우의 이메일 주소와 인증 정보 (Supabase로 관리). 핵심 기능 사용에 계정은 필수가 아닙니다.
  • 구독·결제: Stripe가 처리합니다. 저희는 구독 상태만 전달받으며, 카드 번호를 보거나 저장하지 않습니다.
  • 헬프데스크: 앱 내 헬프데스크 채널 사용 시, 입력한 메시지와 익명 식별자만 지원 서버로 전송됩니다. 헬프데스크는 로컬 데이터에 접근할 수 없습니다.
  • 선택적 통계: 계정 화면에서 명시적으로 동의한 경우에만 회사·직함 정보를 수집하며, 언제든 동의를 철회할 수 있습니다.

3. Google 사용자 데이터

Google 계정을 연결하기로 선택하면 Be.Ark는 Gmail(읽기·발송), Google Drive(읽기 전용), Google Calendar(읽기 전용) 접근을 요청할 수 있습니다. 이 데이터는:

  • 사용자 기기에서 로컬로만 처리됩니다 (다이제스트 표시, 요청한 문서 읽기, 메일 초안 작성)
  • Be.Ark 서버에 업로드·저장되지 않으며, 저희가 열람하지 않습니다
  • 광고에 사용하지 않고, 판매하지 않으며, AI 모델 학습에 사용하지 않습니다
  • 발송은 사용자의 명시적 승인이 있을 때만: 모든 발신 메일은 건별 확인 단계를 거칩니다

OAuth 토큰은 기기 내 로컬 볼트에만 저장됩니다. Be.Ark에서 연결을 해제하거나 myaccount.google.com/permissions에서 권한을 철회하면 접근이 제거됩니다.

Be.Ark의 Google API 정보 사용 및 전송은 제한적 사용(Limited Use) 요건을 포함한 Google API 서비스 사용자 데이터 정책을 준수합니다.

4. 외부 AI 제공자 (BYOK — 내 키 가져오기)

Be.Ark는 로컬 모델만으로 완전 오프라인 동작합니다. 사용자가 본인의 API 키로 클라우드 모델(예: Anthropic, OpenAI, Google)을 연결한 경우, 해당 대화 내용은 사용자의 기기에서 그 제공자에게 직접 전송되며 그 제공자의 약관을 따릅니다. Be.Ark 서버는 이 경로에 없습니다.

5. 보관 및 삭제

  • 로컬 데이터: 항상 사용자의 통제 하에 있으며, 앱에서 삭제하거나 데이터 폴더를 제거하면 됩니다.
  • 계정 데이터: 계정 유지 기간 동안 보관합니다. 계정 및 서버 데이터 삭제는 be@ark.works로 요청해 주세요 — 30일 이내에 완료합니다.
  • 헬프데스크 기록: 지원 제공에 필요한 기간 동안만 보관합니다.

6. 하지 않는 것

  • 개인정보를 판매하지 않습니다.
  • 광고를 게재하지 않으며, 데이터를 광고에 사용하지 않습니다.
  • 사용자 데이터로 AI 모델을 학습하지 않습니다.
  • 서비스 운영에 필요한 위 수탁사(Supabase, Stripe) 및 법적 요구 외에는 제3자에게 개인정보를 제공하지 않습니다.

7. 아동

Be.Ark는 만 14세 미만 아동을 대상으로 하지 않으며, 해당 데이터를 고의로 수집하지 않습니다.

8. 변경

본 방침이 변경되면 이 페이지에 게시하고 상단의 시행일을 갱신합니다. 중요한 변경은 앱 내에서 공지합니다.
