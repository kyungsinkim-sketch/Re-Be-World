주요 기능

단순한 챗봇이 아닙니다. 유저와 함께 성장하는 파트너입니다.

로컬 우선 · 프라이버시

강력한 로컬 모델을 오프라인에서 구동합니다. 대화와 파일은 오직 내 기기에만 저장되어, 민감한 클라이언트 자료도 안심할 수 있습니다. 원한다면 내 API 키를 연결해 사용할 수도 있습니다.

지속되는 기억

대화가 바뀌어도 프로젝트와 사람, 지난 결정을 모두 기억합니다. 매번 처음부터 설명할 필요 없는, 진짜 장기 기억입니다.

나를 닮은 에이전트

내 문서 형식과 말투, 판단 방식까지 학습해 점점 나를 닮아갑니다. 결과물을 넘어 생각하는 방식까지 닮아가는 것 — 크리에이터만의 진짜 경쟁력입니다.

리빙 파일 (Living Files)

노트, 기획서, 초안까지 — AI가 실제 문서를 나와 함께 편집합니다. 내 작업물과 AI가 하나의 파일을 공유합니다.

👥

멀티 에이전트

여러 에이전트에게 역할을 나눠 맡기고 서로 협업하게 합니다. 상위 에이전트가 하위 에이전트에게 일을 지시하는 팀워크까지 가능합니다.

📊

문서·PPT 로컬 생성

제안서와 요약본, 발표 자료까지 내 컴퓨터에서 바로 만들고 PDF로 내보냅니다. 클라이언트 템플릿에 딱 맞춘 창작 워크플로우입니다.
